Description

The Alabama Department of Conservation and Natural Resources (ADCNR) Outdoor Alabama application – the only place for official information – is a powerful tool for hunters, anglers, and outdoor enthusiasts in the State of Alabama, providing easy access to information and services. This application features a digital license display, harvest reporting, rules and regulations, season dates, bag limits, feeding times, and much more.

Features of this app include:
• Access to all current hunting/fishing licenses on your mobile device
• Access to official ADCNR information
• Add and edit multiple weather locations
• Report harvests for both Game Check and Snapper Check

This application requires data access for some features.
