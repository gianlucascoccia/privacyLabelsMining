Great Game!

To the producers of Giant Rush: you guys did an awesome job creating this game! I downloaded this game a couple days ago and I was instantly addicted! It’s one of my favorite games to play! I love all different skins and things you can earn. It makes the game more interesting and more exciting! I also think the levels themselves are great because there are lots of obstacles things to move around. I like that the game gets gradually harder as I play and isn’t the same old same old. The one problem I have with the game is the ads. I’ve never really minded watching ads. It only bothers me when the same ads keep popping up, or they are strange and/or inappropriate. In this case, there is one very repetitive ad that probably isn’t great for kids. Overall I enjoy the game and I hope to be able to play more games like this one in the near future! Hope this was helpful! Jesus loves you! Have a great day! 😊

Great game, but repetitive

This game is really fun and addictive. I could spend a lot of time playing it, and it is real smooth and enjoyable. My only problem is that after a while, it gets kind of repetitive. It is really just the same stuff over and over again. I wish there was more to do it in. It is also pretty easy. It can be annoying sometimes when I don’t get all the people, so I usually replay the level. But other than that, there isn’t much of a challenge in it. Those are my main 2 problems with this game. But other than that, this game is awesome, and I recommend it.
