Description

The Matrix eServices Mobile application is exclusively for the use of Matrix Absence Management, their brokers, clients and the client’s employees.

The mobile application allows users to access claim information and interact with the examiner on the go.

Key Features

• Create an Account
• File a claim
• View claim status
• View claim details
• Report intermittent absences
• Call your examiner
• Email your examiner
• Complete new claim initiation surveys

And for Employers …
• Dashboard reporting and analytics
• Real Time eComm search
• View employee claim status/details
• Email employee’s examiner
• Call employee/s examiner
