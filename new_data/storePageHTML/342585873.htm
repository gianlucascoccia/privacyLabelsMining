Thank you for creating a very helpful app.

I use this app almost daily! Love it!

This is a great app. I use it in so many ways. I have 3 daughters and every time they go on a date I background check the guy. (My girls would be horrified knowing that) but, you can never be too careful. I also like to use it to check who’s calling me. There are so many scams out there and this can bring up a name linked to a number. The deep web search is a great tool to check on your own info to see if it’s compromised. If you’re looking for someone, this is the app to use! You can find out who owns a certain property, social media sites, emails, relatives and associates, all with address and contact info. So far, almost all of the information I’ve found thru this app has been 100% accurate. I was able to find my friends son who was adopted, in one search. It really is a great app for anyone to use.
