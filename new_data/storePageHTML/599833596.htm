Description

Tempo, the premier metronome app for iOS is now available as this free Lite version. Tempo was featured as "New and Noteworthy" and "Staff Favorite" by Apple on the main iTunes landing page. Now you can enjoy Tempo's core features before upgrading to its full suite of features.

Tempo Lite's engine is written from the ground up for high accuracy and stability. It keeps going even in device lock mode, making it a useful aid for live drumming.

As a special treat, we've included the Tracker Module which helps in your practices by tracking the number of bars or time played.

As opposed to purchasing Tempo, upgrading Tempo Lite nets you a dark colour scheme for the flipside views (instead of light).

Visit www.frozenape.com for demo videos.

Features Summary

Configuration
• 8 different time signatures: 1-4/4, 3/8, 6/8, 9/8, 12/8
• 6 rhythm patterns for simple meters and 3 for compound meters
• Customize accents or turn beats off to create more complex rhythms
• Tap tempo function
• Tempo ranges from 10 to 800

Display
• Visual feedback in the form of pulsating LEDs
• Two pendulum types, each with two animation types
• Flash whole screen

Tracker and Automator Module
• Track number of bars played or time your practice
• Coach Mode alternates between unmuted and muted bars.
• Stop playback automatically when limit reached
• Automatic adjustment of tempo every n number of bars or n amount of time

Sound
• Choose from 4 sound sets: Digital, Analog, Blip and Wooden
• Volume control and Mute button

Miscellaneous
• Classical metronome and Italian tempo markings
• Multi-tasking support: plays in background with audio from other apps
• Two interface themes: Light and Dark

I’ve been using this for years.

I’ve been using this for 5+ years and it works great. Lots of amazing features and super user-friendly.

A note for developers: The compound time signatures could use some work. I tend to use different accents for meters such as 12/8 or 9/8 and I wish it would let us use any type of grouping that we want. I also think there should be an option for x/16 time signatures such as 9/16, 13/16, etc.

Some Tuna , 10/08/2021

They all click…

All metronome apps click. This app, Tempo, has a few added features that help me with practice: You can display an indicator for each beat in a measure. E.g. for 4/4 time you see four dots that flash in sequence. The second is the ability to auto increment the bpm. For example increase by 2 bpm every 30 sec.

It also has a feature to make presets that define a starting bps with an auto incrementor. Unfortunately there is no way to edit the preset. So if you gradually improve and want to change a preset from 100 bps to 110 bps for example, you can’t! You need to make a new preset. I hope this a bug or a bad UI that hides the ability to edit a preset, but in either case it mars a nice app.

Pacosa cattan , 11/27/2019

I’ve been using the app for a month now. I picked it because of the integration with the Apple Watch and now I’m going to try the paid version mainly because of the playlist mode for I gig I have.
