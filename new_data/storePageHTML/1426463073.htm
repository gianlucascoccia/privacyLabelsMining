Rextanka , 03/12/2021

Great for tracking, utterly useless for anything else

This app is great for tracking but fairly useless for anything else.

DHL did not test this app, the UI is inconsistent and badly designed.

Details:

Cut and paste is not supported.

You can’t copy a waybill number from the top of the tracking details page. Selecting text is disabled in that view.

In the “in transit” details subview on the “to me” view, if you try to select the waybill number it takes you to tracking details. Even though there is a link highlighted on the right. Really it should allow the waybill to be selected to allow a text cut operation. It should change view only if the link is selected via a touch event.

You can’t paste anything in the chat “how may we help you” text box (so if you want to supply a waybill number accurately you’ll need to flip between this app and somewhere else, or write it down and attempt to enter it accurately which is hard to do on a small keyboard)

If you have a problem with customs clearance (and since DHL ships a lot from Europe, I this is likely) this app is horrible.
