Description

Private Wi-Fi protects a user while using un-encrypted Wi-Fi hotspots. With Private Wi-Fi VPN, the app will provide a highly secure connection when connected to an un-encrypted hotspot.

Wi-Fi alerts will alert a user when they are connected to a Wi-Fi hotspot that is under attack.
