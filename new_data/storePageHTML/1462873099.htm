Description

GPS Converter is an essential, an advanced, yet simple to use mobile application which is designed to convert geographic coordinates to/from different types of coordinate systems. It provides more options than any other GPS Converter out there!

Easily and quickly convert coordinates from/to:
==================================
- UPS/UTM using WGS 84
- MGRS using WGS 84
- Latitude Longitude Decimal Degrees
- Latitude Longitude Degrees Minutes Seconds (D°M'S")
- Latitude Longitude Degrees Minutes (D°M')

Save and recall:
============
Save and recall frequently used coordinates.

Share coordinates:
=============
Share coordinates via Notes, Messaging, E-mail, Twitter, and other App extensions.

GPS Coordinate Pro (IAP required to unlock some items):

- Unlimited Pin Save
- Unlimited GPS Conversions
- No Ads
