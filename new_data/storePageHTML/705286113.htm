It’s great but could use some tweaking

I absolutely love having something that I can let my kids watch when I need them to have some down time and that we can use as part of devotion or Bible study time. However there are a couple things that could make this app better. First, the screen orientation on my ipad is only on portrait so I either have to type with the screen sideways if I want to use the keyboard or use the touch screen. Second the search always comes up with no results even when I type the name of a show that I know is available. It would be able to search a keyword and have shows or episodes pop up that contain that topic. For example we have been talking about wisdom and when I search online for shows on JellyTelly about wisdom I get a list, but when I search from that list on the app I get no results. Those things won’t keep us from continuing to use the app but they sure would make it a little easier.
