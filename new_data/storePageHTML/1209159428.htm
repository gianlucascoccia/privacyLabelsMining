Description

*** Save 20% off Inventory List Lifetime License until the release of version 3 (coming soon) ***

Inventory List is the best way to manage your inventory and keep track of what you have and where. You can customize it, share it, and finally stop using spreadsheets!

ADVANTAGES:
• You no longer have to do manual inventory
• You can track all of your stock across multiple locations
• Add the details that matter to you with custom fields
• Fewer errors than using a spreadsheet
• Calculate your exact current inventory value
• Invite other people to work with you
• Backup & Sync via iCloud

POPULAR USES
Inventory List has powerful features. Popular uses include Stock Management, Asset Tracking, and Inventory Control.

STOCK MANAGEMENT
• Out-of-stock indicators
• Multiple prices
• Custom fields
• Set minimum stock threshold
• Move stock between locations

ASSET TRACKING
• Multiple images
• Date reminders
• Custom Filters
• Tags and custom filters
• Track multiple barcodes/codes

INVENTORY CONTROL
• Collections
• Locations
• Custom filters
• Global search
• Collaboration

KEY FEATURES:

IMPORT
Import your existing inventory in A CSV file and get a headstart.

STOCK INDICATORS
Quickly identify which items are out of stock. You can also set a minimum stock level and see when stock is running low.

CUSTOM FIELDS
Add information like warranty details, additional barcodes, colors, etc.

CUSTOM FILTERS
Combine multiple tags and use rules like ANY or ALL

BARCODE SCANNER
Lookup items by scanning their barcode or any custom code

COLLABORATION
Invite collaborators to work on your inventory. They can use their device and existing Apple ID.

WORKS OFFLINE
Handy when you need to make counts in a stock room without a network connection.

AND MUCH MORE
Move stock, templates, CSV export, flexible organization, global search.

(^)> , 06/22/2018

Great Ideas. Clunky UI

This is the app I want to use. Of the inventory apps I’ve tried, I think this has the most potential.

As of v2.0.4 there are a few issues, namely when I try to send an issue to the developer from the app I can’t. My guess is the develop is relying on Apple’s mail app being installed, but it doesn’t have to be, so in my case, I can’t even send feedback. I have found a few reproducible crashes I’d like to report.

That said, I’ve purchased the subscription to this app as the developer is doing frequent releases to address issues.

The UI interactions are not ideal and it took me a while to start to understand how to use the app. For example, “Collections” i think took me a good 5-10 min of fumbling around to find all the editing controls, which leads to UI complaint:

The “More” section, those are buttons in that horizontal scroll, not purely informational. Also you can had multiple tags to an item in one go, and you can add an item to multiple collections, but not in one go. You have to go back and forth to do it.

I still like this app, the rate of updates shows the developer is currently dedicated to making a great app. It’s got issues, nothing that can’t be fixed with time. More importantly, this app knows it’s voice. It’s not mature yet, but it’s clear where it wants to go.

A solid 3 stars, it’s not great or fantastic yet, I think it will get there though with the right support.
