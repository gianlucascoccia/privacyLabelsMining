Okay I guess...😐😐

I mean look it’s not exactly a terrible game but it definitely isn’t good either. There are sooooooo many ads. I swear every time you finish a level, ad. Next level, ad. So annoying!! This game is also so boring! Some people might like this game but I do not. All you do is use a scoop and drag cubes to the bottom of the screen. This is such click bait meaning #1. No magnet at the bottom and #2 the scooper is so small. It takes so long to scoop up all the cubes in the first place. The scooper is also super glitchy so it’s really annoying to control. But now onto the positives... the levels are super creative with the designs. As you get further on there are obstacles however I found them more frustrating than challenging, only because the scooper kept getting caught. The scooper does get bigger for bonus rounds making it easier to collect cubes. All in all I gave this game 3 stars because it really does depend on the person however I personally do not recommend this game. Like I said there are positives and negatives this game just was not for me, and has improvements to be made.
