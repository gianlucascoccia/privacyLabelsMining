Description

◆100万ダウンロード突破！AI英会話・英語スピーキングアプリ◆
AIキャラクターを相手に英語を話せるから
気を使わずに、時間や場所も自由に英語力を鍛えられる！

▼従来の英会話・英語学習のこんな悩みを解決▼
『緊張する・恥ずかしい』 => AIキャラクター相手に自分のペースで英語を話せるので失敗しても大丈夫！
『予約・通学が面倒』 => あなたの都合ですぐに英会話・英語スピーキングを学べる！
『自分に合った英語勉強法がわからない』 => 最適なカリキュラムをアプリが提案。発音等も分析！

▼「第二言語習得理論」に基づいて、インプットからアウトプットまで段階的に英語学習をサポート▼
（インプット）
1. 単熟語学習：会話前に単熟語を予習
2. リスニング：会話の流れを音声で理解
↓
（アウトプット）
3. 会話：例文を読み上げて英会話・英語発音の練習
4. 英作文：キーフレーズを使って自分で英作文
5. 応用練習：キーフレーズを他の場面で反復学習

▼アプリならではの豊富なコンテンツ▼
感情豊かなAIキャラクターたちとストーリー形式で楽しく英会話・英語学習！
定期的なアップデートでストーリーはどんどん追加されていきます！
・ビジネス英会話、日常英会話など合計700個以上のレッスンで英語力を伸ばす！
・旅行、電話、買い物、会議など様々なシーンで英語フレーズ・英会話を体験
・ネイティブがよく使うイディオムや身近なトピックであるニュース英語を学べるスペシャルコンテンツも！
・AIキャラクターと自由に会話ができるAI英会話「バディと仲良くなろう」や、その月に世の中で話題になっている事柄について会話する「マンスリーコンテンツ」など、期間限定コンテンツを毎月配信！

▼あなたにあった毎日の英語学習をサポートする機能が充実！▼
・あなたの英語レベルをAIがズバリ判定！「レベルチェックテスト機能」
・あなたの英語レベルに合わせた毎日の学習カリキュラムをAIが「自動作成」
・毎日の努力がひと目でわかる「学習記録」
・他のユーザーと競ってレベルを高めていく「ランキング」
・絶対に忘れたくない英単語や英語フレーズを保存しておく「ブックマーク」

▼AI英会話とは？▼
音声認識・合成音声により、人ではなくAIキャラクターと対話する新しい英語・英会話学習です。
機械学習や自然言語処理、ディープラーニングなどのAI技術によって英作文や発音の採点が可能に。
