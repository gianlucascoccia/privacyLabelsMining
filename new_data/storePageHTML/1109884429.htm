What’s New

May 6, 2019

Version 1.0.1

Thanks for using MLG Soundboard Ultimate Memes! We’ve made a lot of changes since the first release.

•Favorites Menu
If you hold down a sound you can now favorite it in order to shortlist it for the future.

•Main Screen Rework
Now, you can scroll vertically through all your memes with larger tiles. We’ve also made some UI tweaks to make it look dank.

•Ads Nerf
We’ve fixed a bug that served an ad on the first click, which was too early in the experience. We’ve also added a survey feature in order to disable all ads for 2 hours, even if you leave the app and come back later.

•Dank Menu
Our new Dank Menu allows you to control the speed, pitch, and sound behavior when the silent switch is enabled. Here you can disable ads and also request memes. We consider all requests regardless of rating and we will add the best ones in our next update.

•Fixed memory issues
We’ve investigated and fixed a memory leak that led to crashes of the app over prolonged use. The app should perform a lot better now with these improvements.

That’s all for this update!
