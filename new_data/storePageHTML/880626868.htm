Description

Take the popular audio-visual instrument on the go with you!

Patatap is a portable animation and sound kit. With the touch of a finger create melodies charged with moving shapes. Warning: contains flashing images.

This is the official offline port of the popular website http://www.patatap.com/

——————————
By buying this app you're supporting the makers who publish and host this website to millions of people to enjoy for free.

Thank you for your support and happy music making!

What’s New

Version 0.0.3

• Sound plays from speakers even in Silent Mode
• Higher resolution animations
• Adapts to all screen sizes
• Keyboard compatible
• Enables external control by MIDI controllers like OP-Z or DAWs like Ableton
——————————
• Stopped relying on Cordova for Embedding
• Using latest WKWebView interface directly in SwiftUI to load website locally
• Works correctly offline, including Airplane Mode
• Automatic MIDI input listening through CoreMIDI
