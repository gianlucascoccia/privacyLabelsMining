Does not Respect the Notification settings in App

I enjoy the refreshed UI and other necessary functionalities organised nicely.

But my major complaint at the time of writing this review is, “This app does not respect the notification settings which I have set inside the application”. Publisher of this app sends overwhelming number of promotional push notifications at different inconvenient times, which is okay. But it should also follow the notification settings I have set inside the.
If I have to disable push notifications from the device settings what’s the point of having that settings in the first place.
Also I don’t want to disable all sort of notifications. Few categories are valuable to me.

So I would appreciate if the publisher takes some time and fix that inconsistency.

Thanks for the refreshed app.
