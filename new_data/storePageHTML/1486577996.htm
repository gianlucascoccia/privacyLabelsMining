Description

Beat Blade is an exciting music runner game that offers stunning neon levels and 2021 hit songs. Simply using one thumb to control the character, you can chop the blocks and dodge the traps, reaching the end of the track.

Runner + blade + music + neon = WOW!

Key features:
1. Distinct color lighting effect for every track
2. Wide range of popular songs including EDM, Hip Hop, KPop and Anime
3. Extraordinary cube slashing visual effect
