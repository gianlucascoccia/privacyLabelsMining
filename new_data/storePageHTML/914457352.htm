Description

Take full control of your photo's metadata.

*Note: Remove Metadata, Date Edit, and Location Edit are premium features, require one-time “Unlock Everything” in–app purchase.

Metapho is a simple and clean viewer for photo metadata such as date, file name, size, camera model, shutter speed, and location.

Metapho Action app extension lets you check metadata of a photo, and export the file without metadata, from anywhere iOS share sheet is available.

• Viewing Metadata •
- Browse all your photos in iCloud Photo Library
- See essential EXIF data of a photo
- Get photo location and lens direction on the map
- View raw metadata in dictionary format, which can be copied to clipboard
- Apply filters to your library such as favorites, location-ness, and creation dates

• Editing Metadata •
- Safe Share: Remove geotag and personal data before sharing
- Remove Metadata: Erase/Restore metadata without duplication
- Date Edit: Change date and time of a photo
- Location Edit: Add or change geolocation data of a photo
- Batch Operation: Edit multiple items’ metadata at once
- Multiple Exporting Options: Save as a new copy or replace original with a new version

• Files Support •
- Add a bookmark of a file or a folder in Files app
- Remove or change image metadata of files directly with editing in-place
- Read and write items in folders shared by Files app such as iCloud Drive, external storage, SMB server, and more

Metapho also supports Live Photos and videos.

*Note: Remove Metadata, Date Edit, and Location Edit are premium features, require one-time “Unlock Everything” in–app purchase.

Still worth keeping around, even with iOS 15

When iOS 15 first came out and I saw it had native metadata viewing, I naively thought I could delete Metaphoto (have been using for years). Apple’s implementation is much worse. For starters, it only works in Photos. Compared to Metaphoto that works anywhere there’s the share button (I use it in Messages all the time).

Apple also frequently has the focal length field blank for photos imported to my library from my canon, even while Metaphoto can accurately show me all the fields.

Metaphoto is simply a must-have for any photographer.
