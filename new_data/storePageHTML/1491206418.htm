To many freezes

Ok grabbed this game surprisingly from an advertisement and thought that something different. To my amazement I really really really like the game but progress is hindered because the app randomly opens advertisements while carving then I get a open failed error and have to restart. I have uninstalled about 5 times now with the same result. My next issue is on some advertisements that just don’t allow you to go back in the game without closing.

Like I said I like the game and so far things move fluently. I am not a wood turner but this game is calming I think because you concentrate to reproducing the shape you see as close as possible. Which at least helps me relieve stress.

In all the game would be a 5 star because I have nothing to compare it to that I have played but the constant restarts are annoying.
