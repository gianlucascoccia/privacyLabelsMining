Description

▷ iLocation is a tool to search and share location easily and quickly.

With iLocation, you can tell the current location or the specified location by several screen clicks. And if you want to know the coordinates or address of the current location, you can simply run iLocation.

When you want to explain an appointment, or when you need to know your current location in a hurry, you can also pin the map on top of the map and simply select the transport to share the location.

◉ Function

• Automatically display current location information when running App
• Coordinate and altitude display of current position or specified position
• Show the location's address
• Show compass bearing
• Change the map format (General, Satellite)
• Specify the center of the map and check the coordinates and addresses
• Specify coordinates by entering coordinates
• Specify an arbitrary location by entering an address (Pro)
• Save map to camera roll or clipboard
• Save your location to Favorites for reuse
• Save the specified location information and map or share it via SNS
• The location is sent to the URL so that the recipient can connect to the map immediately

If you have iLocation, you will be able to explain your location without having to talk.
