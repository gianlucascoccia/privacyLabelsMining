No I wont give a name. , 11/28/2020

I have faith in mobile games again

I used to play more mobile games when they costed a few bucks, were actually engaging, and were not designed from the ground up for micro transactions. This game fits all three criteria’s. This game is even more fluid than Mirror’s Edge: arguably the most revolutionary free running game ever created.

Only requests I have:
a) BIG ONE: PLEASE make a small micro transaction so that users can disable ads. I love how this game is micro transaction free but that would be a transaction I’d pay for since it doesn’t affect gameplay.
b) you cannot run alongside ramps. It is sorta inconsistent since you can run on all other solid surfaces. Implementing that would be cool, even if going on the side of the ramp does not give you the same momentum that running on the ramp would.

Sammarki , 11/04/2020

Almost there

I have been playing for a while now. I’ve the devs listen to player feedback and have been improving the game a lot! It’s still one of the best feeling parkour games out there and with the concept of multiple totally viable paths even as early as level two with different risk and rewards, it feels very free and fun. On a few occasions I wish I had 360 degrees of turn to maybe back track to avoid enemies or recover from a fall, but that is just about my only complaint. The occasional 3rd person view and slow motion shots takes away from the immersion a little, but that isn’t something I’d fight to remove. Points for style.

Nathanael Sanborn , 12/10/2020

It’s not finished

Having played this game for a day, I’ve encountered many problems. First of all, it’s buggy. Very buggy. I’ve had many times where I glitched inside buildings and got stuck on tables and being able to perform multiple exploits. This game needs to be looked at and improved on in terms of stability. My second biggest problem...too many ads. I understand many free games have ads because well...they’re free. However some games like this one have an ad for what sometimes seem like after you run 5 feet. It’s very annoying.

There are a few things a like about this game though. I love the idea, it’s much more interactive than most games. It requires attention. Fast reflexes. It’s very enjoyable, or the idea of the game is.

Please fix this game because I can see this being a very fun game to add to any library.
