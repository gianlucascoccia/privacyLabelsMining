Description

This app provides a comprehensive dictionary of English roots and the reverse dictionary

●● Root - meaning dictionary
- Over 4450 entries of roots, prefixes and suffixes with detailed meaning and explanation

●● Meaning - root dictionary
- Over 4000 entries of meaning and the corresponding roots
- For example: if you look up “good”, you will find “bene-” and “eu-”. Both roots mean “good”

●● Topics
- Entries are divided into topics for easy learning
- Some topics are living things, body parts, colors, dimensions, direction, nature, location, math, sensing, shape, time, etc.

● group
- create your own group
- put an entry into your groups

● note
- write note for each entry

● flashcards
- use flashcard to learn the roots and meaning

● Basic features:
- search feature
- bookmark feature
- built-in dictionary: long press a word to look up

● If you want to give feedback, please use the feedback feature in the settings
