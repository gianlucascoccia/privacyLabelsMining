Description

Soccer Stat Recorder 3000 is the most complete statistic and playtime tracker for parents, managers, coaches. Join the community! The app is fully featured in free mode, but does have ads. Purchase the subscription to remove ads for the best experience.

Features:
* Stat Manager:
- Predefined Stats (40+)
- CUSTOM Stats
- Select which Stats are shown in charts, menus and emails
- Subtypes allow for even deeper insights(example: Shot Types: Header, Volley, Half Volley, Shot)
* Player Manager:
- Load Players
- Images, Birthdate
- Player / Parent Contact Info
- Quick Text Message, Call and Email Buttons
- Default Positions / Jersey Number
- Guest Player Support
* Team Manager:
- Store Coach Information
- Store Managers Information
- Quick Text Message, Call and Email Buttons
- League Information, Team Formations, and Team Colors
* Schedule
- Create Game Schedules
- Create Practice Schedules
- Repeating events
* Position Manager:
- Track playtime against the 1-11, or create your own positions
* Formation Manager
- Drag and drop players to positions.
- Different Formations for different age groups predefined
- Create your own formations!
* Track Stats!
- Fast and easy UI to track goals
- Shot locations
- Assists
- Shot on and off target
- Corners
- Saves
- Crosses
- Passes (completed and missed)
- ….
- and many more for individuals and teams.
* Track Possession:
- Quick Toolbar allows people to quickly track / pause possession time.
* Game View:
- Timers for Total In/Out of Game
- Timers for Time since last Sub In/Out
- Guest Player Support
- Fair Playtime Calculated Per Player / Position
- Customizable / Sortable Player Tracking interface
- End of Game PK Logger
* Longest Passing Streak:
- Now when you tap the possession button to gain possession, and tap it again to count passes.
* Notes:
- Take Notes about a particular game event and have that show up in the log
* Game Clock:
- Games can be set to have an incrementing clock or decrementing playtime clock.
- We now support Clockless mode.
* Review / Export Game Info:
- Edit past games
- Review past games
- Store Notes about the game
- Export past games summaries
- Formatted data file in PDF, Excel, Plain text, and CSV format.
- Max Preps Output File
* Export:
- Individual Season Stats / Complete Game Log.
- We support Maxpreps formatting as well.
* Season Filtering:
- Assign a season to a game and filter your games and stats on those seasons!
* Import teams via Excel.
- Check out the FAQ for how!
* Twitter Integration
- Swipe Right on Game Events to create a tweet!
* Location Manager
- Manage the locations you visit most in an easy selector
* Opponent Manager
- Manage Teams you play against
- Store Contact, Email, and Notes

Why a subscription app? (1) We do a lot of continuous work on this app. (2) We respond to emails all the time about how to use the complex features of the app (3) We hate ads (4) We needed a way to let people try the app without buying it initially.

Love the app - iCloud issue fixed

Update: (5+) First, the support just rocks! Thanks the quick response. There are 17 players and 1 coach who are going to be excited. Second, this app is worth at least $20. Anyone who is serious about improving youth team and player development needs this app. Finally, see the pros/cons below, but the best feature is the customization. Very well thought out.

Pros — Overall this app is great. It’s easy to navigate, intuitive and very customizable. Quick and easy to learn and when configured to the detail I needed, it made keeping stats the coach needed simple. Much better than SoccerMeter (which is dead anyhow). But still fantastic for youth league play, which is fast.

Cons or added function - First, the possession buttons could be larger. Having an audible prompt for a pass or possession would make a great add. Helps with confirming when you are watching/covering a youth game, as possession changes so fast and often. Using the screen real estate for common functions instead of player avatars would be better.
