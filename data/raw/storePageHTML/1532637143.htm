Description

Battery Widget & Usage Monitor App allows you to monitor your phone usage such as battery status, storage usage, memory & space, and provide general device information.
Beautiful widgets for iphone iOS 14 home screen allows you to view system info!

► BATTERY
• Graphical battery level display
• Battery state: discharging, charging, full
• Low-power mode

► BRIGHTNESS
Graphical Screen Brightness level display

► STORAGE
• Real time storage usage monitoring
• Graphical display of Used and Free disk capacity monitoring

► MEMORY
• Real time memory usage monitoring
• Visualization of Active, Inactive, Wired and Free memory allocation

► NETWORK
• Detect Wi-Fi network & Cellular data network connection statuses.
• Detail Network Interfaces

► HARDWARE
• List display of Hardware information

► DEVICE INFO
• General device information such as Device Name, System version, System Uptime and Boot time.
• Detail device information

Data insights and Operating system information will be updated in the upcoming versions.

You can freely edit the widget theme colors that fit your iOS14 home screen the most!
