Description

This is the Pro version of HolyBible K.J.V.
Main features:
- Read King James Version Bible
- Highlight verses
- Add verses to bookmarks
- Daily verses
- Add note to verses
- Change settings about font type and sizes
- No Advertisement
- Lots more features to discover.

1. Ads Free
2. Completely same UI design and navigation as Free version


Features in this new version:

1. Nav as books/cap/verses.
2. View by books/cap/verses.
3. Search by book name.
4. Add it to your favorite.
5. Use the selector to choose which you like.
6. Remove ad in it.
