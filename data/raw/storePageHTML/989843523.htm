chad_fitz , 11/13/2018

Few missing features, Compression seems inefficient

I really like this app overall—it has lots of features and allows conversion from and from numerous formats. The user interface is also really good. Now, I'll list concerns/missing features in order of importance:

1. One additional feature that I would really appreciate is the ability to set the output specs of the GIF during the save process, for example: resolution, frame rate (I realize this can be set globally, but I would very much prefer to be able to manipulate it during the output process), index palette (number of colors, perceptual, adaptive, custom, web, etc—similar to the GIF options in Adob Animate), and dithering (error diffusion, noise, halftone, none—again, similar to Adob Animate).

2. Output GIF compression seems off (possibly due to background choices being made in '1.' above). When comparing the output file size to GIFs generated with other tools, it seems really large on high or medium quality (and there is a huge and noticeable drop-off on low).

3. It would also be nice to be able to adjust things like sharpness/noise, contrast, brightness, saturation/vibrance, shadows/highlights, tint/temp,

4. Frame-by-frame GIF editing, ideally as layers similar to using GIMP.

Thanks again for a great app, and I hope to see continued development.

Developer Response ,

Hello. Thank you for using the ImgPlay and we appreciate your detailed feedback. Haver you ever tried our updated version(newer than v5.0, currently v5.0.3)? Please check our comment as following:
1. In the updated version, you can change frame rate and index palette. We will consider to add other output spec too in the future.
2. We improved our GIF compression on the updated version. Also, next version, you will be able to save the file in the desired size. We will keep improving the compression.
3. In the updated version, you can adjust the effect such as brightness, contrast, saturation, warmth and sharpen!
4. In the updated version, you can delect the frame or reorder the frame. But we do not support the layer feature yet. We will check if we can add this feature in the future.
Thank you so much for your feedback 👍
