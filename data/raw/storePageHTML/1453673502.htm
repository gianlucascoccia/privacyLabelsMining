I_like_soccer , 08/04/2020

Pro/Rel

I really like this app, it’s super easy to use, very fun. There’s only one problem with the Pro/Rel system, and that’s if I get to the end of a season with a tournament with multiple leagues designed for pro/rel, the app will ask where I want to send promoted or relegated teams, but then it doesn’t do it. There’s no way (that I can find at least) to reset the tournament for another season and actually let the app promote and relegate the teams it asked to promote and relegate. The idea that it ASKS which leagues I want to send teams to but then doesn’t have a way to actually DO that and reset the tournament is just so weird to me, so maybe I’m just missing it somewhere? I’ve looked all over the app, all over the app website and App Store description of it, but I can’t find anything on this. If this is already an option somehow, it would be nice if it were easier to find, or if it’s not an option at all, then obviously it would be nice to have that lol. Other than that, it’s beautifully designed, the logos are really good looking, and the league system setup is the most detailed of any tournament app I’ve ever used.

Developer Response ,

Hi, thanks a lot for using "Winner - Tournament maker app" 🏆🥇⚽. The promotion and relegation system currently works between phases of the same tournament, e.g. qualifying from a league phase to a playoff phase of the same tournament. However we will definitely extend the feature to support promotion and relegation between divisions in the upcoming versions of the app. If you have any other ideas for improvements, please let us know at: il.talent.apps@gmail.com or via the "Feedback" button on the side menu of the app.
