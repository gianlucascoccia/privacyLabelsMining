Description

An audio coloration utility plugin featuring individual modes for classic emulation of some iconic music production tools. This tool is meant to bring character, or "dirt" to your incoming audio signals. Save your own presets for instance recall of the sound you've created to find the "sweet spot" in your instruments and/or mixes.

Lo-Fly Dirt installs as an Audio Unit effect for hosts that support the format such as Audio Bus, Beatmaker 3, Garage Band, Cubasis, AUM, etc.

Default Mode:
Adds subtle character of the input sound. Simple saturation and light, preset compression are added to the overall sound bringing out unique tones & textures of your source audio. The wet/dry knob at the 100 or full position would leave source audio "warmed up."

60 Mode:
This preset mode is based on some of our favorite characteristics of an iconic drum machine. Activating this button should invite you to 12 bit nostalgia immediately boasting tasteful saturation, small amounts of compression, and a more rounded high end. Great on drums.

404 Mode [Green Light]:
One of our favorites, this mode is the only mode that should add subtle low-pass filtering to the input signal. This mode will apply low pass filtering at 12db per octave around 11.6Khz. The wet/dry knob will continue to filter the sound to no more than 200Hz. The signal will benefit from some additional signal boosting with minimal make-up gain and small amounts of bit rate reduction, or "dirt."

404 Mode [Orange Light]:
This mode brings the coveted vinyl simulation to Lo-Fly Dirt. Mix knob will tastefully add intensity to your track with it. Try using this mode on an aux/bus channel and send select sounds through it. Very cool stuff!

1200 Mode:
Also built around our love for another iconic drum machine, this mode brings great punch, character, and dirt to any signal it treats. Especially useful for drums and percussion based audio.
more punchier variation of the 60 mode. Enjoy 12 bit goodness by default, but additionally a boost the signals character by our dirt and boost in the low-mid frequencies.

8 Bit Mode:
Simply put, this mode brings back the classic sound of a legendary game console. We've added a bit of punch to the incoming signal and you're able to blend it with great 8 bit artifacts that will allow for some brand new creations...and fun!

SK- 5 Mode:
Also built off the things we love about the classic 8 bit sound, but with a twist...this mode adds HP filtering and punch! Instant dirt is applied when this mode is activated, high pass filtering and punch is applied as you move the signal from dry to wet. The HP filter is set to cutoff at 300Hz.

aeonism215 , 05/24/2020

Love this app, but recent updates seem buggy

The newest version of the app added a bunch of presets, and while the idea seems great, they haven’t been super useful, at least for me.

What’s worse though, is that it seems to have broken the save state features. When I save a session in BeatMaker 3 (can’t speak for any other DAWs) I often return to sessions later, and find that Lo-Fly will have reset itself to its default setting (which is now a very unpleasant highly distorted preset).

This is bad enough when you’re using one instance on a master buss, but when you have multiple instances, going to recover each instance can be a headache. I’d gladly give up the added presets to just be able to save my sessions and have Lo-Fly remember what I dialed in the next time I open the session up. I’m assuming this is a Lo-Fly problem because BM3 has no problem opening up other plug ins with my settings recalled.

Other than this issue, it sounds great, don’t make beats without it, haven’t done so in probably 2 years (100s of beats). It’s an absolute staple that just needs some fine tuning.

Sounds good. Does NOT crash apps. Still needs more integration.

Really cool sound. I have been wanting lo-fi sampling ability and this app brings it.

Wrote in previous review that it crashes apps, but I was mistaken. (Sorry, devs!) The samples I was using were simply too big and that’s what caused the problem.

Would like to see this able to be used in an effect slot in AudioShare. However, devs informed me that AudioShare has to support AUv3. I have given the app 5 starts because it does exactly what it claims to do and also because the devs are so communicative. They address issues quickly and politely.
