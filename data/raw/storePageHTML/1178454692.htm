Syrena Hokura , 12/10/2018

Good app but a few flaws

Okay so I like this app, it is really useful when combining things from videos (I used it to put together different parts of a song I made) that being said, there was a bit of a learning curve, but after a few frustrating minutes I figured it out. However, quite a few times while I was editing my song, I was kicked off the app. This would have been only a minor inconvenience, if only it saved progress after exiting the app. So I ended up having to start all over again, not once, but SEVERAL times🙄

The ads are troublesome and annoying, but I found it easy to get used to them popping up every few minutes or so.
Overall, I would recommend this app, but it would be great if the developer could fix those issues.

lt_jenna47 , 04/17/2017

Ehhh. It's good, but it has some negatives.

In my honesty, I think this app is really cool. Especially how you can download music, edit it, listen to it, and edit it how it looks whenever you're listening to it. For example, like how you can put a picture for it, or how you can put lyrics for the song. It's really cool how the this app has this. Unlike other apps.

However, there are some negatives about it. When you're editing your music, there isn't any undo button. And also, it crashes whenever you try to choose a picture. :/

There are several things that this app could have or improve on. Such as at least a guide or help button to show the user how to edit their music. This is because when I first used it, it was a little confusing and the user might give up trying to use this app and see what else it provides. To add on, it needs to fix its current negatives. Such as crashes and such.

But besides all of my reasoning, I think the app is cool. ^-^
