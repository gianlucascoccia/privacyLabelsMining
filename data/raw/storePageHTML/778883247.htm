DESCRIPTION
-----------------
It's how notes on iOS should work. A unique swipe-gesture manages task lists and adds formatting in a snap. One-tap reminders ensure that you revisit a note later. Folders & favorites keep you organized, while Dropbox syncs notes between devices.

Once you see how productive your everyday notes can be, you'll wonder how you lived without UpWord!

FEATURES
-------------
• Swipe List: efficiently create lists, cross off as done, and highlight as important, all with a swipe
• One-Tap Reminders: faster than ever to save a note or task for later, includes repeating reminders
• Repeating reminders*
• Organize with folders & shortcuts*
• Sync notes across devices with Dropbox
• Optimized for iPhone AND iPad (universal)
• Save to Evernote
• Multi-level undo & redo
• Multiple font sizes
• Cloud backup, version control, and retrieve deleted notes via Dropbox sync (all free)
• Opens straight to a new note to capture ideas faster (turn on in "Settings")
• TextExpander support
• Markdown preview mode
• Full-text search (titles & note contents)
• Word count
• Export to pdf, email, text, or even facebook/twitter
• Landscape typing
• Full suite of x-url-callback actions
• Night mode, for typing in the dark
• Link mode makes phone #'s and website url's tappable
• Sort files by chronological or alphabetical order
• Archive keeps your workspace clean without permanently deleting
• TaskPaper features like @done, tab swipes, and .taskpaper extensions

** For the tech-savvy who were wondering, UpWord uses UTF-8 encoded plain-text with Unix style line breaks. Our plain-text files will work with any modern text editor, so there's no format lock-in.

flcowboy7 , 05/24/2019

99% Perfect Notes App

Have used upword notes for a few years now (bought IAP once, totally worth it). I recently tried a bunch of other apps, none of which hold a candle (in my eyes) with upword. Not all of them are bad, but this one has all the right features for ME and works as intended. The only thing it’s missing is the ability to save a note offline (local to the device). I love in a semi-dead cell area, so when I’m without coverage I can’t access my notes. Unfortunately, most apps don’t offer this function. DropBox does, so the work around is to find the note in DB and save it offline from there. I can then view the note from DB when offline, but can’t edit it.

I wish upword would add a way of saving notes offline and allowing us to edit those notes while offline. Then have them upload and sync to the server when connection resumes.

nursesrus , 07/27/2018

Helpful organizing tool

I use this app everyday and it is great. I use it to make shopping lists at the different stores I shop at as well as for making packing lists for trips and “to do” lists. It is great to be able to mark through an item and then unmark it. One thing I have found “on accident” that I don’t like is that if you scroll to the top of your list too quickly it will “clean up” your list by deleting everything you marked off. I don’t like this feature because when I make a packing list for example, I mark off the things that I have packed and then usually, when I am getting ready to come home from my trip I go back into my list and make sure I pack everything back up to come home. If I have accidentally scrolled to the top of my list too fast it will delete all of the items that I have marked through and that leaves me at risk for forgetting something.
