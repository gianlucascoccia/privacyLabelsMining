Description

The awesome reminder + timer app that is alarmingly great!

• Pop-up reminder alerts with robust repeat scheduling, flexible snooze and full customization.
• Pop-up timers with custom messages, countdown / count up, timer queues and more.
• Support for both timed and location reminders.
• Use Siri to create reminders and import from the Reminders app into Alarmed.
• iCloud syncing & backup. (Extras Package)
• Notes-as-checklist instantly transforms notes into actionable checklists.
• Categories to help you organize your reminders and timers.
• Over 140 high-quality custom sounds included.
• Are you alarmed?

KEY BENEFITS AND FEATURES

Pop-Up Reminder Alerts
• Create a simple reminder with just a couple of taps.
• Want options? Alarmed offers full configurability, including robust repeat options, snooze, choice of custom sounds, pre-alarms, notes and more.
• Both timed and location reminders are supported.
• Drop-down panel for easy access to snoozing and notes.
• The Nag-Me feature allows for auto-repeating of reminders at regular intervals for a fixed period, or until you mark it as being done, whichever comes first. Helps to ensure that you get it done!
• DayMinders (Extras Package) features repeating reminders throughout the day, such as start at 9:00, repeat every 45 minutes for 8 hours.
• Import from the built-in Reminders app. Allows you to use Siri to quickly create reminders while on the go.
• Full sharing support (mail, message, print, AirDrop, etc.)
• Never forget something important again!

Pop-Up Timers
• Simple to create and use - tap, spin and go.
• From 1 second to 999 hours.
• Supports countdown, count up and timer queues
• Pause & resume.
• Super size timers for easy visibility.
• Countdown timers for the usual timing events, such as cooking, laundry, etc.
• Count Up timers for tracking progress on activities.
• Timer Queues are multiple timers in a single entity that fire in sequence.

Plus...
• Includes over 140 custom sounds. From practical to fun to whacky.
• iCloud syncing keeps your data updated on multiple iOS devices as well as providing an off-device backup (Extras Package).
• Supports Interactive Notifications, which allow you to mark reminders done, snooze reminders, restart timers -- directly from the notification (lock screen, Notification Center, Apple Watch).
• Quick and easy to use with many advanced options which are there if you need them, out of the way if you don't.
• A built-in help system so you get the most from Alarmed.
• Database backup / restore (automatic backups on device, can email for safekeeping).
• Full VoiceOver Accessibility support.
• A Universal App with full native support for iPhone, iPod touch and iPad.

~~~ Are you Alarmed? ~~~
