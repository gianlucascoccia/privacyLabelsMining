Description

◆100万ダウンロード突破！AI英会話アプリ◆
AIキャラクターが会話相手だから
気を使わずに、時間や場所も自由に学習！


▼従来の英会話のこんな悩みを解決▼
『緊張する・恥ずかしい』 => AIキャラクターが会話相手なので失敗しても大丈夫！
『予約・通学が面倒』　　 => あなたの都合ですぐに英会話を学べる！
『自分に合った勉強法がわからない』　　 => 最適なカリキュラムをアプリが提案。発音等も分析！


▼「第二言語習得理論」に基づいて、インプットからアウトプットまで段階的に学習をサポート▼
（インプット）
1. 単熟語学習：会話前に単熟語を予習
2. リスニング：会話の流れを音声で理解
↓
（アウトプット）
3. 会話：例文を読み上げて会話の練習
4. 英作文：キーフレーズを使って自分で英作文
5. 応用練習：キーフレーズを他の場面で反復学習

▼アプリならではの豊富なコンテンツ▼
感情豊かなAIキャラクターたちとストーリー形式で楽しく英会話！
定期的なアップデートでストーリーはどんどん追加されていきます！

・ビジネス英会話、日常英会話など合計370個以上のレッスンをご用意！
・旅行、電話、買い物、会議など様々なシーンでの英会話を体験
・ネイティブがよく使うイディオムや身近なトピックであるニュース英語を学べるスペシャルコンテンツも！
・AIキャラクターと自由に会話ができるAI英会話「バディと仲良くなろう」や、その月に世の中で話題になっている事柄について会話する「マンスリーコンテンツ」など、期間限定コンテンツを毎月配信！


▼あなたにあった毎日の学習をサポートする機能が充実！▼
・あなたのレベルをAIがズバリ判定！「レベルチェックテスト機能」
・あなたのレベルに合わせた毎日の学習カリキュラムをAIが「自動作成」
・毎日の努力がひと目でわかる「学習記録」
・他のユーザーと競ってレベルを高めていく「ランキング」
・絶対に忘れたくない単語やフレーズを保存しておく「ブックマーク」


▼AI英会話とは？▼
音声認識・合成音声により、人ではなくAIキャラクターと対話する新しい英会話学習です。
機械学習や自然言語処理、ディープラーニングなどのAI技術によって英作文や発音の採点が可能に。
