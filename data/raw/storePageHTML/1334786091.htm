Experience the ultimate in home automation control with Myro:Home for Leviton/HAI automation controllers. Myro:Home combines Leviton/HAI's proven home automation controllers with the latest features of iOS like FaceID/TouchID, Siri and WatchOS. Finally, a user-friendly and convenient home automation interface for your Leviton/HAI system that is as usable from anywhere in the world as it is from the comfort of your couch.

Anything from changing the thermostat setting to controlling lights or the security system is only a few taps away. The interface is customizable using a "Favorites" tab, where any controller object can be added for quick access and the "Recents" tab provides access to the last used items. iBeacon and Location triggering is also possible.

The "Status" tab gives you a quick overview of the system's status including the security status for each area, the thermostat statuses, auxiliary sensor statuses, as well as any unread messages or zones that are not ready.

The "Security" tab, conveniently organized by area allows you to view all zone statuses, bypass zones or arm/disarm the system (or set the mode for Lumina systems). The "Security" icon at the bottom of the screen displays a count of any insecure or bypassed zones. A filter bar at the top also allows you to quickly find which zones are not ready.

The "Climate" tab provides a list of all configured thermostats and any auxiliary sensors, displaying their status, temperature and humidity levels. Tapping any thermostat allows you to have full control over its functionality including temperature and humidity setpoints, mode, fan and hold.

The "Control" tab allows you to control all of the configured lighting units in your home.

The "Audio" tab is displayed only on panels with an audio interface connected and allows control of all defined audio zones and sources, including any of its metadata!

The "Messages" tab provides a list of all system messages and allows you to display or acknowledge any unread messages.

The "Cameras" tab allows viewing of an unlimited number of MJPEG and RTSP capable cameras. It can also access cameras with static JPEG images and automatically refresh them at 1 fps.

These are just a few of the things you can do with Myro:Home, download and try it all out for yourself!

Hardware Requirements:
Myro:Home supports all recent Ethernet-enabled HAI home automation and security controllers, including: Lumina, Lumina Pro, OmniPro II, Omni IIe and Omni LTe. Controller firmware 3.5 or later is required.

Privacy:
Myro Control, LLC does not use or collect any of your home usage data, private keys or connection details. They are only stored locally on the iOS secure keychain. The communication between the Leviton/HAI controller and Myro:Home directly flow between your device and the controller using an encrypted protocol. Myro Control, LLC does not store any data in the cloud or on their systems at any point.
