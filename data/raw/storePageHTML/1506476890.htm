Mike Thain , 09/08/2020

Fail

This seems like it would be an ok tool for drafting if it wasn’t so buggy that it doesn’t work at all. For 2 of my drafts, when the draft order got set, the app would not pull the new draft order. It just kept a random draft order there, even when I deleted the app and reinstalled. So I manually added the draft order. Then in my last draft the same thing happened, but my league had changed from 8 teams to 10. The app still showed 8 and there was no option to manually add teams. Yahoo sets the draft order 30 minutes before the draft and I was there maybe 15 minutes before. By the time I deleted the app and reinstalled, and clicked through all the settings, and ultimately it wouldnt even let me add the league back because it said the league was already loaded. Then I gave up and just drafted without it. So if you want to use this, make sure you wait until minutes before your draft when the draft order is set before you activate your league for the first time. Don’t load it early and run a mock draft or to look at player values. All that dummy data will stick in your iPads memory with no way to reset it. Also, I almost forgot, for kickers it only counts field goals, not extra points. Kickers don’t matter that much but come on, extra points are easy to count. They have 2 values, EPC and EPM (extra points completed and made???) but EXM is always 0 for every kicker.

Bartguy , 08/25/2020

Still the best, but a few issues this year

Been using this app since the beginning, always with great success. This year’s version, unfortunately, seems less rock solid than previous ones. Just doing mock drafts I’ve had trouble switching between positions - with the app stuck highlighting the position I’m trying to switch to, but not actually switching. This is worrisome if you’re under a time limit to make your pick. I’m using a recent (but not the newest) iPad Pro. There doesn’t appear to be any way to report the issue since the help link just bounces you to a FAQ page.
