engine sounds , 02/26/2021

Room to improve

Cool app, very user friendly and simple. But at the same time that might be the downside of it for some users, I am missing a few things.
1. I would really like to have the option to customize and gradually adjust individual parameters, and play around with custom sound fixes.

2. I would like to be able to gradually adjust existing sound fixes, and be able to make them stronger then they are right now, so I don’t have to use the same sound fix multiple times on the same video.

3. The audio fix seems to remove high pitched whining sounds, which are not bad noise and I don’t want to remove them. I don’t know how the program works, but those whining sounds increase or decrease gradually, like a nice curve. The bad noise doesn’t do that, bad noise is irregular and not gradual like that. Maybe it would be possible for the program to detect the difference.

4. I would really like to have an equalizer.

5. The way you load new videos with the refresh button is a little weird but ok, a simple + and automatic refresh would do it if you ask me.
But it seems that it is buggy too and often I have to restart the app to load new videos. Issues are either that it doesn’t ask me to load new videos, or that it doesn’t want to refresh after selecting new videos.
The first time after starting the app it seems to always work.
