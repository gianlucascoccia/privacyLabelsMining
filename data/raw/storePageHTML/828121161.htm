Description

Dry Practice Drill is the best dry practice timer for the the iPhone or iPad. It is simple to use, and wrapped in a beautiful interface.

The app includes all the drills from Front Sight's handgun training course. but the drill list is yours to customize. Add and remove drills, or tweak times as your skills improve.

For each drill you are given a few seconds to prepare, then an audible "Ready" call. After a random pause, a short tone marks the drill start. You should complete the drill before you hear the whistle.

Use the strings feature to automatically decrease drill time as you practice.

We Include basic instructions how to complete each drill. This is especially helpful for complex drills like malfunction clearing.

This is the perfect companion for your trip to Front Sight, and a great way to keep the momentum going afterwards.

Hint: If you cannot hear the sounds, check your device's mute switch. ;-)

Good, but could be better.

The app is good and has what it absolutely needs. But there are three features that would help it be much better.

First is that it really needs to disable the automatic screen lock when running. Alternately it not pausing when the screen locks would probably be sufficient.

Second is that it would be real nice if the timer didn’t start with the reset time. For example practicing quad loads, reset takes involves unloading and putting everything back on my shell carriers. I need 10-15 seconds for that but I’m always fully reset prior to hitting the go button for a new set and that just wastes practice time. Making it toggleable would be good. But if I have set up to do 5 strings, I only really need three resets.

Third is the random start. It would be best if you could contain the maximum and minimum delay.

Developer Response ,

Thank you for the review and the great suggestions! Your first suggestion is included in the most recent release (v3.1.0).
