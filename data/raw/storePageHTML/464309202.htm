C W

This app is helpful in the way it will tell you if you have numbers. The only thing is that is does not tell you how many number or what prize you may have won if any. This is a feature I think would improve the usage and popularity of the app. As for the scanner feature of this app, why take a picture? Can't an actual scanner that a lot of users have be used by the app to achieve the scanning process? It all feels so cumbersome an ancient. So if these changes are possible, please do it. By-the-way, how the drawing goes for the gift cards is interesting. The person with the most points get the gift card? Will the points be returned if you are not the winner? If not then why participate. That needs to be refined a bit and then it may catch on even more with everyone. Idea is good, execution poor. Seeing the lottery drawing and news stories are great, as well as, notifying public that there is a winning ticket not yet collected. Excellent! All and all not too bad. Always room for improvement.
