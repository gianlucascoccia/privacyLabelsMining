Description

Reverse Lookup is a FREE reverse phone lookup app for iOS devices that retrieves all available information, including the name, associated with a phone number. This app is a handy tool to identify unknown callers. Currently, it works for cell, landline, toll-free, and fax phone numbers from the United States and Canada.

FEATURES:
- Finds name associated with most numbers.
- Very simple, responsive, fast, and intuitive.
- Copy/Paste numbers from the call log.
- Report suspicious phone calls.
- Completely free!

To use this app, you have to enter a valid 10-digit phone number, then lookup that number. Reverse Lookup will query the latest phone directory and retrieve information associated with that phone number. This app also ensures that it has the most recent and reliable data for that particular number before displaying the result.

Hopefully, you will find this phone number information finder tool useful. Please help grow this app by sharing it with your friends and family.
