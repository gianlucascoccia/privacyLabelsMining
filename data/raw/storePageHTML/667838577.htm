Description

This App will send push notification alerts to your device when defined price levels are hit.

Real-time quotes are used.

Alerts can be set using the Bid or Ask as the trigger price.

Swipe gestures are used to adjust price levels quickly and easily.

Alerts can be easily sorted, modified, deleted or moved.

Ideal for technical traders who are waiting for key price levels to be reached

Upgrading to the Full Version of this app offers the following additional benefits:

- Unlimited number of alerts
- Custom alerts sounds
- Long duration alerts sounds
- Additional Forex pairs (30+)
- Custom text messages

The Full version supports the following currency pairs:

EURUSD, GBPUSD, AUDUSD, USDJPY, USDCHF, USDCAD, EURAUD, EURCAD, EURCHF, EURGBP, EURJPY, GBPJPY, NZDJPY, AUDJPY, CADJPY, NZDUSD, XAUUSD, XAGUSD, GBPCHF, USDMXN, EURMXN, USDSGD, USDHKD, USDTRY, EURTRY, AUDCAD, AUDCHF, AUDNZD, CADCHF, CHFJPY, EURNZD, GBPAUD, GBPCAD, GBPNZD, USDCNH, USDNOK
