Description

Read script, lyrics & speech in an easy to use teleprompter with mirroring, speed change and font change

With this app, you can read your script easily without worrying about forgetting the next line. Teleprompter works well on iPhone, iPad and iPod touch too!

It has the following features:
- Create and edit your scripts on the device
- Import Txt, Word and PDF files from the cloud
- Record Videos inside of the app
- Set the speed of the playback
- Set a specific time for the playback
- Mirror the playback both vertically and horizontally
- Set the font size
- Use your Bluetooth keyboard as a remote to control​ the playback
- Customize keyboard shortcuts


So many more features are coming soon!
If you have some ideas on​ how to improve it, please email me@matekovacs.com.

Works great, no gimmicks or nonsense

I’ve tried a TON of these apps and this is by far my favorite. Excellent at doing what it does simply and effectively. Best interface I’ve seen for this from a free app (w/watermark). Stable and flexible.

My biggest suggestion! Please include some sort of number or indication for speed other than the very small drag-bar that is in there now. The bar is very small and dragging the circle a tiny bit has a big impact, and I want to know what speed to use so I can set it precisely and not have to fiddle around with it every time.

Thanks for making this great app!

Deppisch , 11/19/2019

Spend more on a better app.

I was on a shoot yesterday using this app. I paired my phone as a remote and it worked well. Since I was using a proper teleprompter mount, I tried to mirror the text, and it asked me to upgrade. Fine, that’s expected. I started the trial, and then the remote wouldn’t work—an error popped up saying I had to go pro to use it. But that’s what the free trial enables. It also worked fine on the free version earlier, so this was really confusing. So I upgraded to pro, and it still didn’t work. Every time I tried to pair an iOS device to use as a remote, I got the error. I could control my iPhone from the iPad, but it would give me the error the other way around. I turned the iPad on and off, closed the app and restarted, deleted and reinstalled, but I had the same issue each time. So I deleted it and downloaded another app that actually worked. I’m out the $4.99, and I got to look like an idiot in front of a client. But I learned a lesson: test all of the functionality of an app before the day of the shoot. Because normal, vital features might inexplicably get nerfed once you pay the developer.

Sparkey0512 , 05/07/2020

It’s fine for new YouTubers, but not very good.

The app was fine to get me started on my channel, but as soon as I can find something better I’m ditching this app. It is annoying that I have to pay $5 a month to use an app that’s not very good. Controlling the scrolling speed is way too frustrating. The scrolling speed is either too fast or too slow. The dumb slider is too coarse and should be designed better. There is no reason that it could not be a numeric speed to one decimal place. Also linking one device to control another device locks up regularly while you are trying to record. You import a .rtf file but the app is very specific about where it can locate it. The app has a built in editor but it is very basic. Don’t even try editing on a phone in landscape mode. Overall I will be ditching this app as soon as I can find something better.
