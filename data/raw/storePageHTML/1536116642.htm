Really good for a new webtoon app

So I know this app is still pretty new, and in my opinion it’s not bad for a new webtoon app. I really like the subscription system and I wished the other webtoon apps like Tappytoon, tapas, and LezhinComics had something similar to that. However, I did notice that some of the webtoons that are available in these other platforms are a little behind in updates on Manta. Like for example on Manta a certain webtoon would still be on hiatus while on the other apps, it has already started updating again. Also, since this is still a new app, the webtoon library is very limited. I am noticing that there is now a ‘Coming Next’ page so I’m excited for that. Overall I think this app has a lot of potential to be big, but to do that the webtoon library would need to be a lot bigger.

I look forward to future updates!
