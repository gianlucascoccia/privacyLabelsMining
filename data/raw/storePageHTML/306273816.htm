Suggestions on improvements

Hi, I appreciate this App, and like it very much. I have the following suggestions on improvements for your consideration:
(1) Suggest to add a function: when tap the top of an opened document (maybe the document name), it will automatically bring the document to the bottom line. If would be helpful when the document is really long.
(2) Suggest to NOT alway popping out a dialogue box to choose “save” or “not save” when closing a document, IF the document was only opened and viewed without any revisions or updates. The save/not save options should be only given when a document was modified.
(3) Suggest to add a function of “bullet” for editing the content of a document.

disgruntledavidreader:( , 02/26/2020

Italics not converting correctly

I’ve been using this app for about a year now to revise writing projects on the go and work on new chapters. So far, it’s been really helpful. Normally, when I’m ‘done’ with a document or want to work on it on my laptop, I’ll move it from the app to my laptop using iTunes, then open the document and copy/pasta the work I’ve done into the version on my laptop. Before, this was a very smooth process, aside from some issues I had with formatting, but I figured out a way to get around that. So, like I said, copy/paste was smooth for a while, but lately words typed in italics in the app have been showing up in documents on my laptop either in a different font or just funny looking (not sure why), so I’ve had to manually find each italicized word after copy/paste, delete them, and then retype them using italics on my laptop. Other than this new problem, the app is great for my needs, but it would be awesome if this was fixed.
