Description

Employees can enter hours to a timesheet, view previous time in summary, view leave balances and see timesheet related pending tasks on an easy to use native mobile app. Supervisors can view employee timesheets, Approve employee timesheets and view pending approval tasks on an easy to use native mobile app.

When paired with a Costpoint 8 upgraded system, employees will have access to expense functionality which will allow them to capture expense receipts with ICR, edit and claim outstanding expense, create, edit and submit expense reports and Supervisors will be able to approve expense reports, expense charges and expense attachments in an easy to use native mobile app.
